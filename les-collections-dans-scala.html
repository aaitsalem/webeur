Les collections en scala

1- Définition

Nous allons vous présenter les collections les plus utilisées avec quelques exemples expliquant leurs utilisations, si vous souhaitez plus d’informations vous avez la documentation scala sur les collections ici : https://docs.scala-lang.org/overviews/collections/introduction.html

Nous distinguons deux types de collections en scala : les collections immutables et mutables

Collection immutable : Une telle collection ne changera jamais après sa création. Par conséquent, vous pouvez compter sur le fait que l'accès à la même valeur de collection à plusieurs reprises à des moments différents donnera toujours une collection avec les mêmes éléments. Impossible de modifier une des valeurs de la collection, il est seulement possible de les lire. Scala propose toujours par défaut la version immutable

Collection mutable : Est connue pour avoir certaines opérations qui modifient la collection en place. Donc, traiter une collecte mutable signifie que vous devez comprendre quel code modifie la collection

Ex : collection mutable
val list = scala.collection.mutable.List (1,2,3,4,5)
list+=6 //add 6 a la collection
Ex : collection immutable
val list = scala.collection.immutable.List (1,2,3,4,5)
list+=6 //erreur de compilation

Si on veut ajouter un élément à cette collection, faudra créer une nouvelle collection

val list = List (1,2,3,4,5)
println (list) // =List (1,2,3,4,5)
val list2 = 6::list
println (list2) // = List (6,1,2,3,4,5)

2- Les collections


Tuple

Represente un couple de valeurs

def addMult ( x:Int , y : Int ) : Tuple2 [Int,Int] = Tuple2(x+y,x*y)
val tuple = addMult(10,20)
println (tuple) // (30,200)
println (tuple._1, tuple._2 ) // (30,200)

Map

La Map est une collection contenant un ensemble de clés valeurs.

Par defaut une Map en scala est une collection immutable , cad que vous ne pouvez pas en changer les valeurs directement , si on veut créer une Map mutable faudra a la creation notifier le paquetage scala.collection.mutable

Creation d’une Map immutable

val mapImmutable = Map ("toto" -> 1, "titi"-> 2,"tata"-> 3)
val mapImmutable2= Map (("toto" , 1),("titi", 2),("tata", 3))
println (mapImmutable)
println (mapImmutable2)
println(mapImmutable ( "toto")) //affiche l’element don’t la clé est "toto"

Creation d’un Map mutable

Il faut oligatoirement setter le paquetage scala.collection.mutable

val mapMutable= scala.collection.mutable.Map (("toto" , 1),("titi", 2),("tata", 3))
mapMutable(“toto“)=10
// ou bien
mapMutable+=“toto“-> 10 //ajouter un élément
mapMutable-=“toto“ //enlever un élément

Comme sur les tableaux la modification d’une Map immutable se fait par la création d’une nouvelle Map et dans ce cas faut que la Map soit une variable et non une constante

var map = Map[String,Int]()
map=map+ (("toto" , 1),("titi", 2),("tata", 3))

Itération sur une Map

val map=Map(("toto" , 1),("titi", 2),("tata", 3))
map.foreach(e=>println (e))
for ( e <- map ) printl (e)
Ou bien
for ( (k,v) <- map ) println (k,v)
On peut aussi avec le for / yield pour transformer une Map dans ce cas multiplier la valeur par 2 ou bien avec la foction map
val map=Map(("toto" , 1),("titi", 2),("tata", 3))
val mapInverser = for (( cle,val) <- map) yield (cle,val*2)
Pour recuperer les clés d’une Map ou ses valeurs
val cles = map.keys
val valeurs=map.values
Liste

La scala list contient une liste linéaire séquentielle d'éléments. Dans cette liste, chaque élément doit être de même type.elle peut etre vide Nil

Le premier élément s’appelle « head » et le reste de la liste « tail »

var liste = List(0,1,10, 3,4,5)
println(liste.head) // affiche le premier élément
println(liste.tail) // affiche le reste des élément éléments
println (liste(2)) // affiche 10

Evidement et comme toute collection , la collection List peut contient de nombreuse méthodes de tri , d’inversion de réduction et de transformation

Set

Est un ensemble d’éléments de type Set sans ordre particulier et chaque élément est répété une seule fois

var liste = Set(0,1,10, 3,4,5)

Les mêmes méthodes de la Liste peuvent s’appliquer sur le Set: head , tail ,contains, enlever un élément , ajouter un élément...

println(liste+6) // ajoute 6
println(liste-0) // enleve le 0
